# Portfolio (React + Vite)

Projeto de portfolio em React com Vite.

## Scripts

- `npm run dev`: inicia ambiente local
- `npm run build`: gera build de produção em `dist/`
- `npm run preview`: pré-visualiza a build localmente

## Deploy no Vercel

1. Faça push do projeto para o GitHub.
2. No Vercel, clique em **Add New Project** e importe este repositório.
3. Use as configurações:
	- Framework Preset: `Vite`
	- Build Command: `npm run build`
	- Output Directory: `dist`
4. Clique em **Deploy**.

O arquivo `vercel.json` já está configurado com esses valores.
